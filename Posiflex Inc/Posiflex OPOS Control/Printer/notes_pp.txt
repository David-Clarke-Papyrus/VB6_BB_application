0. Before installation
	Do not install this release over any previous release that may be on 
	the machine being used. If another version of the software is already 
	installed on the computer, please uninstall it using
	the 'Add/Remove Programs' function from the Control Panel.

1. Overview
	We are glad to announce the release of the Posiflex POS Printer
	OPOS Control Module for Windows.
	By using this kit, you can easily build applications
	based on the OPOS Application Programmer's Guide Release 1.8.
	This kit is designed to be used with Microsoft Visual Basic v6.0
	or other 32-bit development environment.

2. Files
	After installation, you have following files:

	POSI.OCX - Posiflex POS Printer OPOS Control Object
	FLEX.DLL - Posiflex POS Printer OPOS Service Object
	POSI.LIB - Posiflex POS Printer OPOS Control Object Import Library
	FLEX.LIB - Posiflex POS Printer OPOS Service Object Import Library
	POSI.TLB - Posiflex POS Printer OPOS Control Object Type Library
	FLEX.TLB - Posiflex POS Printer OPOS Service Object Type Library

	TUNE.EXE - Posiflex OPOS Control Registry Manager
		( This program must be executed after installation )

	Demo.EXE - a Visual Basic 6.0 Demo Executable
		( for default installation path )
	Demo.VBP - a Visual Basic 6.0 Demo Project
	FORM1.FRM - a Visual Basic 6.0 Form for the above project
	OPOSALL.BAS - OPOS 1.8 constants for Visual Basic

	README.TXT - This file.

3. Installation
	To install Posiflex POS Printer OPOS Control Module,
	uncompress setup zip file to a subdirectory,
	run setup.exe in the directory and follow the instructions.

4. Testing
	a. turn on the Posiflex POS Printer which should be connected to the system
	b. after installation, run tune.exe and make appropriate choices
	c. start Visual Basic 6.0 and load demo.vbp, or run demo.exe instead
	d. make any changes needed on the demo project and rerun to test the printer

	Note:	In OPOS Control Manager, you can double click an item, 
		make some changes and hit "Replace" to replace the selected original.
		After finishing all your modifications, please press
		"Finish" button to leave the program, otherwise
		your modifications will not take effect.

5.  Conflicts and limitations

	Must turn printer DIP switch 2-1 off if used with PP-7000II firmware version 6.8.
	To show the printer firmware version,
	please press <Feed> button when turning on the printer.
	
	Might not correctly report printer errors,
	if used with any Windows Printer Driver installed.
	
	This version does not support following features.

		a. DirectIOEvent
		b. ErrorEvent (but will retry printing in async. mode)
		c. DirectIO
		d. RotateSpecial
		e. Width adjustment in PrintBitmap()
		f. ChangePrintSide
		g. ClearPrintArea
		h. MarkFeed
		i. PageModePrint
		j. CompareFirmwareVersion, UpdateFirmware
		k. ResetStatistics, UpdateStatistics
